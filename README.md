
<h2>StapuBox 3 Screen OTP Verification</h2>
<p>This is a mini project that performs OTP verification using user mobile number. With added features like :</p>
<ul>
   <li>10 digit mobile number validation</li>
   <li>Invalid OTP message</li>
   <li>Resend OTP</li>
   <li>Cooldown timer of 60 seconds</li>
</ul>
